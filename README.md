# MiBilletera React 19.2.6

Proyecto React actualizado a 19.2.6, modularizado y preparado para crecimiento.

## Ejecutar

```bash
npm install
npm run dev
```

El comando `npm run dev` levanta el frontend y la base de datos local con `json-server`.

- Frontend: http://localhost:5173
- API local: http://localhost:4000

## Comandos útiles

```bash
npm run dev:front
npm run api
npm run typecheck
npm run build
```

Más detalles en `README_ACTUALIZACION_REACT19.md`.
